Hai, selamat datang di web docs spark bot


INTRODUCTION :


Spark adalah bot berbasis website yang diprogramkan menggunakan bahasa program javascript, bot ini dilengkapi banyak fitur dan saya juga telah memasukkan lebih dari 400+ frasa dan akan terus ditambah.


FITUR :


  • Dapat mengingat nama user
  • Bisa menjawab pertanyaan [non education]
  • Bisa menjawab pertanyaan [education]
  • Beberapa minigame
  • Perhitungan matematis
  • Terdapat pertanyaan fungsional(seperti menanyakan hari, tanggal, jam dll)

INFO : SPARK menggunakan cookie, untuk menyimpan progress user

Fitur lanjutan :


penggunaan fitur perhitungan math :


  • Masukkan angka ke input yang tersedia contoh : 1+1
  • Lalu tekan tombol enter/chat, disamping area input

Keterangan :


  • [*] perkalian
  • [/] pembagian
  • [+] penjumlahan
  • [-] pengurangan
  • [**] pangkat •> 2²
  • [%] persentase bilangan

Game yang telah ditambahkan :


  • cekgay
    NOTE: Fitur ini tidak bermaksud menyinggung siapapun, ini dibuat untuk fun saja
  • GBK [gunting, batu, kertas]
    Bedanya, saya telah memasukkan UI yang lebih baik

  • untuk keluar dari permainan, kamu bisa ketik 'quit'


Rules/aturan penulisan input :


  • Jangan masukkan spasi di depan dan belakang kata



    Anggaplah [-] adalah spasi
  • Anda tidak bisa memasukkan lebih dari 100 karakter pada input
